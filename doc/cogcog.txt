*cogcog.txt*    LLM as a vim verb

INTRODUCTION                                    *cogcog*

cogcog adds LLM verbs to vim. They compose with motions and text
objects. Most verbs run at 0.3s via raw API. Check and execute
use cloud models for depth.

Note: `ga` overrides show-ASCII, `gs` overrides sleep,
`<C-g>` overrides show-file-info.

FAST VERBS (raw API, 0.3s)                      *cogcog-verbs*

                                                *cogcog-ga*
ga{motion}      Explain code. No prompt. Count = verbosity:
                  gaip = concise, 1gaip = 1 sentence, 3gaip = detailed
                Auto-includes visible windows and quickfix.
                Panel open = stateful. Closed = stateless.
                Visual: prompted.

                                                *cogcog-gaa*
gaa             Explain entire buffer.

                                                *cogcog-gs*
gs{motion}      Generate code. Raw API (fast). Output in code buffer.
                Auto-detects language, strips fences. :w to save.

                                                *cogcog-gss*
gss             Generate from entire buffer.

                                                *cogcog-gr*
<leader>gr{motion}
                Refactor in-place. u to undo.

CLOUD VERBS                                     *cogcog-cloud*

                                                *cogcog-gc*
<leader>gc{motion}
                Check with strongest model (opus). 10-90s.
                Split reused.

                                                *cogcog-gx*
<leader>gx      Agent execute. Cloud agent with tools. Runs in
                the context panel. Stderr shows inline as progress.
                Uses COGCOG_AGENT_CMD.

                                                *cogcog-cd*
<leader>cd      Discover project. Strongest model.
                Saved to .cogcog/discovery.md (gf-navigable).

PLANNING                                        *cogcog-plan*

                                                *cogcog-ctrl-g*
<C-g>           Fast conversation (raw API, 0.3s).
                From code: prompts, shows current filename.
                From panel: sends as-is.

                                                *cogcog-cy*
<leader>cy      (visual) Pin selection to context panel.

VIM STATE AS CONTEXT                            *cogcog-context*

                                                *cogcog-gj*
<leader>gj      Jump trail. Last 8 locations. Reuses ask split.

                                                *cogcog-g-dot*
<leader>g.      Recent changes. Reuses ask split.

ga auto-includes visible windows and quickfix.

PANEL                                           *cogcog-panel*

                                                *cogcog-co*
<leader>co      Toggle. Open = ga stateful.
>
                :read .cogcog/review.md     add skill
                :read !git diff             add tool output
                dap                         delete section
<
                                                *cogcog-cc*
<leader>cc      Clear, reload system prompt.

                                                *cogcog-cp*
<leader>cp      Improve prompt from bad response.

                                                *cogcog-cancel*
<C-c>           Cancel jobs. Pass-through when idle.

CONFIGURATION                                   *cogcog-config*

    ga/gs/gr/<C-g>  COGCOG_FAST_MODEL (raw API, fast)
    gx              COGCOG_AGENT_CMD (cloud agent with tools)
    gc/discover     COGCOG_CHECKER (strongest model)

                                                *cogcog-fast-model*
COGCOG_FAST_MODEL   Fast local model for most verbs.

                                                *cogcog-agent-cmd*
COGCOG_AGENT_CMD    Cloud agent for execute.

                                                *cogcog-checker*
COGCOG_CHECKER      Strongest model for check/discover.

                                                *cogcog-backend*
COGCOG_BACKEND      "anthropic" or "openai" (default: anthropic)

                                                *cogcog-api-url*
COGCOG_API_URL      API endpoint.

                                                *cogcog-api-key*
COGCOG_API_KEY      API key.

                                                *cogcog-model*
COGCOG_MODEL        Default model when FAST_MODEL not set.

                                                *cogcog-max-tokens*
COGCOG_MAX_TOKENS   Max tokens (default 8192).

PROJECT FILES                                   *cogcog-project*

.cogcog/system.md       System prompt. Shared by all verbs.
.cogcog/session.md      Auto-saved/restored.
.cogcog/discovery.md    Project map.

QUICK REFERENCE                                 *cogcog-cheatsheet*

Fast (0.3s): >
    gaip / gaa      explain
    Visual ga       ask
    gsip / gss      generate
    <leader>grip    refactor
    <C-g>           plan
    <leader>gj      jumps
    <leader>g.      changes
<
Cloud: >
    <leader>gcip    check (opus)
    <leader>gx      execute (agent)
    <leader>cd      discover (opus)
<
Other: >
    <leader>cy      pin
    <leader>co      panel
    <leader>cp      improve prompt
    <leader>cc      clear
    <C-c>           cancel
