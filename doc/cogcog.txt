INTRODUCTION                                    *cogcog*

cogcog adds LLM verbs to vim that compose with motions and text
objects. Your vim state is context: visible windows, quickfix,
jump trail, recent changes.

Note: `ga` overrides show-ASCII, `gs` overrides sleep,
`<C-g>` overrides show-file-info. Deliberate tradeoffs.

VERBS                                           *cogcog-verbs*

                                                *cogcog-ga*
ga{motion}      Explain code. No prompt — instant. Fast local model.
                Count controls verbosity:
                  `gaip`  = concise
                  `1gaip` = one sentence
                  `2gaip` = clear
                  `3gaip` = detailed with examples

                Auto-includes visible windows and quickfix.
                Panel open = stateful. Closed = stateless.
                Response split is reused across calls.

                Visual mode: select then `ga` — prompted.

                                                *cogcog-gaa*
gaa             Explain entire buffer.

                                                *cogcog-gs*
gs{motion}      Generate code. Agent backend with tools.
                Output in a new code buffer with line numbers.
                Auto-detects language, strips code fences.
                `:w filename` to save. `q` to close.

                                                *cogcog-gss*
gss             Generate from entire buffer.

                                                *cogcog-gr*
<leader>gr{motion}
                Refactor in-place. Fast local model.
                Replaces the code directly. `u` to undo.
                Works on code and prose. Cancellable with <C-c>.

                                                *cogcog-gc*
<leader>gc{motion}
                Check with strongest model. Deep review.
                Response split is reused across calls.

PLANNING                                        *cogcog-plan*

                                                *cogcog-ctrl-g*
<C-g>           Context-aware:
                - Code file: prompts, sends to local agent (has tools)
                - Context panel: sends buffer as-is
                - Exec buffer: continues the conversation

                                                *cogcog-cy*
<leader>cy      (visual) Pin selection to context panel.

VIM STATE AS CONTEXT                            *cogcog-context*

                                                *cogcog-gj*
<leader>gj      Ask about jump trail. Last 8 jump locations.
                Reuses the ask split.

                                                *cogcog-g-dot*
<leader>g.      Review recent changes. Lines you edited.
                Reuses the ask split.

ga auto-includes:
- Visible code from all windows
- Quickfix entries (LSP diagnostics, grep, make errors)

EXECUTE                                         *cogcog-exec*

                                                *cogcog-gx*
<leader>gx      Agent execute. Multi-file work with live activity.
                Uses COGCOG_AGENT_CMD (cloud) or COGCOG_CMD (local).
                Agent stderr shown in the buffer as progress.
                `<C-g>` in the exec buffer continues the session.

CONTEXT PANEL                                   *cogcog-panel*

                                                *cogcog-co*
<leader>co      Toggle panel. When open, `ga` becomes stateful.
>
                :read .cogcog/review.md     add a skill
                :read !git diff             add tool output
                dap                         delete a section
<
                                                *cogcog-cc*
<leader>cc      Clear context, reload system prompt.

                                                *cogcog-cancel*
<C-c>           Cancel running jobs. Passes through when idle.

DISCOVERY                                       *cogcog-discover*

                                                *cogcog-cd*
<leader>cd      Map project by domain using strongest model.
                Saved to `.cogcog/discovery.md` (gf-navigable).
                Options: Open / Update / Re-discover.

PROMPT IMPROVEMENT                              *cogcog-improve*

                                                *cogcog-cp*
<leader>cp      Learn from bad response. Appends fix to
                `.cogcog/system.md`.

CONFIGURATION                                   *cogcog-config*

Four backend paths:

    ga/gr       COGCOG_FAST_MODEL via COGCOG_API_URL (--raw)
    gs/<C-g>    COGCOG_CMD (local agent with tools)
    gx          COGCOG_AGENT_CMD (cloud agent, falls back to COGCOG_CMD)
    gc/discover COGCOG_CHECKER (strongest model)

                                                *cogcog-fast-model*
COGCOG_FAST_MODEL   Model for ask/refactor (fast, local).

                                                *cogcog-cmd*
COGCOG_CMD          Local agent CLI for generate/plan.

                                                *cogcog-agent-cmd*
COGCOG_AGENT_CMD    Cloud agent CLI for execute (<leader>gx).
                    Falls back to COGCOG_CMD if not set.

                                                *cogcog-checker*
COGCOG_CHECKER      CLI for check/discover (strongest model).

                                                *cogcog-model*
COGCOG_MODEL        Default model (when FAST_MODEL not set).

                                                *cogcog-max-tokens*
COGCOG_MAX_TOKENS   Max tokens (default: 8192).

                                                *cogcog-system*
COGCOG_SYSTEM       System prompt text (for shell usage).

                                                *cogcog-backend*
COGCOG_BACKEND      "anthropic" (default) or "openai"

                                                *cogcog-api-url*
COGCOG_API_URL      API endpoint for raw path.

                                                *cogcog-api-key*
COGCOG_API_KEY      API key.

Simplest setup: just ANTHROPIC_API_KEY. All verbs work.

PROJECT FILES                                   *cogcog-project*

.cogcog/system.md       System prompt. Auto-loaded.
.cogcog/session.md      Auto-saved/restored.
.cogcog/discovery.md    Project map from <leader>cd.
.cogcog/*.md            Skill templates.

    gaip / gaa              explain (count = verbosity)
    Visual ga               ask (prompted)

    <leader>grip            refactor in-place
    <leader>gcip            check (deep)
<

    <C-g>                   plan / continue
    <leader>cy              pin selection
    <leader>co              toggle panel
    <leader>gx              agent execute (cloud)
    <leader>gj              jump trail
    <leader>g.              recent changes
    <leader>cd              discover project
