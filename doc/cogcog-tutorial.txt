*cogcog-tutorial.txt*   Getting started with cogcog

TUTORIAL                                        *cogcog-tutorial*

SETUP                                           *cogcog-tutorial-setup*
>
    export ANTHROPIC_API_KEY="sk-ant-..."
    cp bin/cogcog ~/.local/bin/ && chmod +x ~/.local/bin/cogcog
<
Neovim (lazy.nvim): >
    { dir = "/path/to/cogcog" }
<

DISCOVER THE PROJECT                            *cogcog-tutorial-discover*
>
    <leader>cd
<
Strongest model maps your project. `gf` on paths to navigate.

QUICK EXPLAIN                                   *cogcog-tutorial-ask*
>
    gaip            concise explanation (no prompt)
    gaa             explain entire buffer
    3gaip           detailed with examples
    Visual ga       "is this thread-safe?" (prompted)
<
Auto-includes visible windows + quickfix. Split reused. `q` to close.

STATEFUL EXPLORATION                            *cogcog-tutorial-explore*
>
    <leader>co          open panel (ga becomes stateful)
    gaip                first question
    gaip                builds on previous
    <leader>co          close (back to stateless)
<

PIN FROM MULTIPLE FILES                         *cogcog-tutorial-pin*
>
    " file A: visual select → <leader>cy
    " file B: visual select → <leader>cy
    <C-g> → "can these race?"
<

PLAN WITH AGENT                                 *cogcog-tutorial-plan*
>
    <C-g> → "add rate limiting"     local agent reads files
    <C-g> → "use token bucket"
<

GENERATE CODE                                   *cogcog-tutorial-gen*
>
    gsaf → "implement rate limiting"
    gss → "scaffold the module"
    :w src/ratelimit.ts
<

REFACTOR IN-PLACE                               *cogcog-tutorial-refactor*
>
    <leader>graf → "simplify"       u to undo
<

CHECK                                           *cogcog-tutorial-check*
>
    <leader>gcaf                    strongest model reviews
<

MULTI-FILE WORK                                 *cogcog-tutorial-exec*
>
    <leader>gx → "refactor auth"   cloud agent, live activity
    <C-g>                           continue in exec buffer
<

INVESTIGATE                                     *cogcog-tutorial-investigate*
>
    gd → gd → gd       navigate around
    <leader>gj          how do these connect?
    <leader>g.          any bugs in my changes?
<

TEST AND ITERATE                                *cogcog-tutorial-iterate*
>
    gsaf → "implement retry"      generate
    :w src/retry.ts                save
    :make                          test (errors → quickfix)
    gaip                           explain failure (quickfix auto)
    gsaf → "fix it"                iterate
    <leader>gcaf                   verify
<

IMPROVE PROMPTS                                 *cogcog-tutorial-improve*
>
    <leader>cp → "too generic"
<
Appends fix to `.cogcog/system.md`.
