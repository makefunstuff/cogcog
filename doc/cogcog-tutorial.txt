*cogcog-tutorial.txt*   Getting started

SETUP                                           *cogcog-tutorial-setup*
>
    export COGCOG_BACKEND=copilot
    export COGCOG_CHECKER="..."      " optional stronger check/discover cmd
<
Neovim: >
    open any code file and try: gaip
<

EXPLAIN (0.3s)                                  *cogcog-tutorial-ask*
>
    gaip            concise (no prompt)
    gaa             entire buffer
    3gaip           detailed
    Visual ga       prompted question
<

PLAN (workbench + tools)                        *cogcog-tutorial-plan*
>
    <C-g> → "add rate limiting"
    <C-g> → "use token bucket"
    <leader>gy      pin selection
<

GENERATE (0.3s)                                 *cogcog-tutorial-gen*
>
    gsaf → "implement it"
    :w src/file.ts
<

REFACTOR                                        *cogcog-tutorial-refactor*
>
    <leader>graf → "simplify"       u to undo
    large rewrites open review       a apply / q close
<

CHECK                                           *cogcog-tutorial-check*
>
    <leader>gcaf
    uses cogcog --raw unless COGCOG_CHECKER is set
<

PI (other terminal)                             *cogcog-tutorial-exec*
>
    ln -s /path/to/cogcog/pi-extension ~/.pi/agent/extensions/cogcog
    Terminal 1: nvim
    Terminal 2: pi
    <leader>gx        reminder: use pi in the other terminal
<

INVESTIGATE                                     *cogcog-tutorial-investigate*
>
    <leader>gj          jump trail
    <leader>g.          recent changes
    <leader>gQ          review quickfix
    <leader>gR          rewrite quickfix targets
<

ITERATE                                         *cogcog-tutorial-iterate*
>
    :make → gaip → gsaf → :make → <leader>gcaf
<
