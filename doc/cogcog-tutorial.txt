*cogcog-tutorial.txt*   Getting started with cogcog

TUTORIAL                                        *cogcog-tutorial*

SETUP                                           *cogcog-tutorial-setup*
>
    export ANTHROPIC_API_KEY="sk-ant-..."
    cp bin/cogcog ~/.local/bin/ && chmod +x ~/.local/bin/cogcog
<
Add to Neovim (lazy.nvim): >
    { dir = "/path/to/cogcog" }
<

DISCOVER THE PROJECT                            *cogcog-tutorial-discover*
>
    <leader>cd
<
Wait for the strongest model to map your project. Output saved
to `.cogcog/discovery.md`. Press `gf` on any path to open it.

UNDERSTAND CODE                                 *cogcog-tutorial-ask*
>
    gaip            "explain this" (instant, no prompt)
    3gaip           detailed explanation
    Visual ga       "is this thread-safe?" (you choose the question)
<
`ga` includes visible windows + quickfix as context automatically.

EXPLORE WITH CONVERSATION                       *cogcog-tutorial-explore*
>
    <leader>co          open panel (ga becomes stateful)
    gaip                first question
    gaip                builds on previous answer
    <leader>co          close (back to stateless)
<

CROSS-FILE QUESTIONS                            *cogcog-tutorial-pin*
>
    " file A: visual select → <leader>cy
    " file B: visual select → <leader>cy
    <C-g> → "can these race?"
<

PLAN A FEATURE                                  *cogcog-tutorial-plan*
>
    <C-g> → "add rate limiting"
    <C-g> → "use token bucket"
<
Agent reads files to understand your codebase.

GENERATE CODE                                   *cogcog-tutorial-gen*
>
    gsaf → "implement rate limiting"
    :w src/ratelimit.ts
<
Code buffer auto-detects language.

REFACTOR IN-PLACE                               *cogcog-tutorial-refactor*
>
    <leader>graf → "simplify"
<
Code replaced directly. `u` to undo.

CHECK                                           *cogcog-tutorial-check*
>
    <leader>gcaf
<
Strongest model reviews for bugs and edge cases.

MULTI-FILE AGENT WORK                           *cogcog-tutorial-exec*
>
    <leader>gx → "refactor auth across all files"
<
Agent activity streams in real time. `<C-g>` to continue.

INVESTIGATE                                     *cogcog-tutorial-investigate*
>
    gd → gd → gd       navigate around
    <leader>gj          "how do these locations connect?"
<
>
    " edit some code
    <leader>g.          "any bugs in my changes?"
<

TEST AND ITERATE                                *cogcog-tutorial-iterate*
>
    gsaf → "implement retry"      generate
    :w src/retry.ts                save
    :make                          test (errors → quickfix)
    gaip → (quickfix auto)         explain failure
    gsaf → "fix it"                iterate
    <leader>gcaf                   verify
<

IMPROVE PROMPTS                                 *cogcog-tutorial-improve*
>
    <leader>cp → "too generic"
<
Appends a fix to `.cogcog/system.md`. Prompts improve over time.
